# 功能简介
 - 这是一个服务于旅客、售票员、列车管理员的多功能火车售票系统
 - 旅客主要是用来购票及个人信息的维护
 - 售票员用来为旅客提供出票、退票、预定等服务
 - 列车管理员用来对列车车次进行管理
